ai2_kit.feat.spectrum.md_spectra.function_atomic.calculate_corr(A: ndarray, B: ndarray, NMAX: int, window: Optional[int] = None)[source]#

Calculate the correlation function: corr(t) = <A(0) * B(t)>. Here, A(t) and B(t) are arrays of the same dimensions, and A(t) * B(t) is the element-wise multiplication. The esenmble average < > is estimated by moving average.

Parameters:

  • A (np.ndarray, in shape of (num_t, ...).) – The first dimension refers to the time steps, and its size can be different. The remaining dimensions (if present) must be the same.

  • B (np.ndarray, in shape of (num_t, ...).) – The first dimension refers to the time steps, and its size can be different. The remaining dimensions (if present) must be the same.

  • NMAX (int.) – Maximal time steps. Calculate corr(t) with 0 <= t <= NMAX.

  • window (int, optional.) –

    The width of window to do the moving average.

    <A(0) * B(t)> = 1 / window * sum_{i = 0}^{window - 1} A(i) * B(t + i).

Returns:

corr – corr(t) = <A(0) * B(t)>

Return type:

np.ndarray, in shape of (NMAX + 1, …).
