# Spectroscopy Workflow

This example contains a comprehensive workflow for calculating IR, Raman, and Sum-Frequency Generation (SFG) spectra using Machine Learning Molecular Dynamics (MLMD) and Deep Wannier (DW) models. 

## Directory Structure

* **`omb/`**: Scripts and input files for computing Wannier centers and polarizabilities (zero-field and small electric fields along $x$, $y$, $z$).
* **`train/`**: JSON configuration files and datasets required to train Deep Potential models for energy/force, atomic dipoles, and polarizabilities.
* **`simulate/`**: LAMMPS input scripts, data files, and submission scripts for executing the production MLMD simulation.
* **`predict/`**: Scripts to predict atomic dipoles and polarizabilities for every frame along the generated MD trajectory using the trained DW models.
* **`spectra/`**: Post-processing scripts to compute time-correlation functions and generate the final spectra.
* **`run.sh`**: The master bash script orchestrating the execution of the workflow.

## Execution Guide

You can run the entire workflow sequentially by executing the master script:
```
sh ./run.sh
```

## Important Note

This workflow is fundamentally sequential. Because downstream tasks rely on data generated by preceding steps (e.g., prediction requires both the MLMD trajectory and the trained DW models), users are advised to execute each stage in the prescribed order to ensure all data dependencies are properly met.
